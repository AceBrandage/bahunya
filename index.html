Bahunya

10KB classless CSS framework with responsive typography, navbar, syntax highlighting, and much more.

Bahunya is a CSS framework (like Bootstrap) with no classes. This means that as long as you write semantically correct HTML and include the minified (10kb) version of this framework, your website will look like this! Just include the following to your <head> 

<link rel="stylesheet" href="https://cdn.rawgit.com/kimeiga/bahunya/css/bahunya-0.1.1.css"/>

Bahunya will work out of the box on HTML5 compliant websites, but can be easily customized with custom CSS and CSS classes.

Follow along the source code of this page! Press ctrl shift i on Windows to open Devtools on Chromium browsers.

Customization

Navbar

Bahunya has a custom navbar that you can drop in or take out; your preference!

Just drop the following html elements into your header's nav, and it will convert your text-based navbar to a navbar with a hamburger menu like this site. This solution uses no JavaScript, just the checkbox hack

<input type="checkbox" id="menu-toggle">
<label for="menu-toggle">&#9776;</label>

The first element is the invisible checkbox that controls the visibility of the dropdown menu. The second element is the hamburger icon itself, using the popular trigram for heaven symbol

You'll notice that Bahunya's navbar only works with a listless navigation (one without ul and li elements). This is because people with screen readers find it easier to navigate listless navigation as the screen reader peruses it quicker than a listful (?) navigation.

Your markup should look like the following. An <h1> containing an anchor tag for the site title, and then the anchor tags for the links and the hamburger menu. 

<body>
  <header>
    <nav>
      <h1><a href="#">BAHUNYA</a></h1>
      <input type="checkbox" id="menu-toggle">
      <label for="menu-toggle">&#9776;</label>
      <a href="#">STARTUP</a>
      <a href="#">MINIMAL</a>
      <a href="#">BOLD</a>
      <a href="https://github.com/Kimeiga/bahunya" target="_blank">GITHUB</a>
    </nav>

Syntax Highlighting

Bahunya uses Prism JS for syntax highlighting within <pre><code> tags. Prism requires a CSS file and a JS file to operate. The CSS file is included in Bahunya (it's tiny), but the JS you'll have to import. Just add the following to your <body>.

<script src="https://cdn.rawgit.com/Kimeiga/bahunya/26313661/prism.js"></script>

In order to activate Prism on a pre-code tag, add one of the following classes on the <code> tag, not the containing <pre> tag.

HTML (Markup)
class="lang-markup"
CSS
class="lang-css"
C-like
class="lang-clike"
JavaScript
class="lang-javascript"

To get more languages, head to Prism's download page, select more languages, and then include the provided JavaScript file in your website (instead of Bahunya's)

Bahunya uses the Okaidia theme which is based on Monokai. It's the smallest Prism theme in filesize at 1.75KB

Colors and Fonts

Why a dark scheme?

Why does Bahunya use white text on a dark background? Recent extensive German opthamology research concluded that reading dark text on a light background thins the choroid, the vascular layer of the eye, contributing to myopia, while reading light text on a dark background increases the thickness of the choroid. As such, Bahunya ships with dark text on a bright background for opthamological safety, but feel free to change it.

Primary Color

Bahunya uses #E7BF60 gold as a primary color. To change this color to what you like, use the following CSS

:root {
  --primary: #YOUR_COLOR;
  --primary-dark: #YOUR_COLOR_BUT_DARKER;
}

input[type=submit],
button[type=submit],
button:not([disabled]) {
  background: var(--primary);
}
input[type=submit]:hover,
button[type=submit]:hover,
button:not([disabled]):hover {
  background: var(--primary-dark);
}

dd + dt {
  border-top: 1px solid var(--primary);
}

blockquote {
  border-left: 1px solid var(--primary);
}

mark {
  background: var(--primary);
}

a {
  color: var(--primary);
}

::-webkit-scrollbar-thumb:hover {
  background: var(--primary);
}

That should restyle all the elements that had the primary color on them.
